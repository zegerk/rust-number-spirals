# rust-number-spirals

Create SVG [prime number spirals](https://en.wikipedia.org/wiki/Ulam_spiral), ported the code from Python to Rust using chatGPT.

Demo: https://number-spirals.zeger.app/

## Compile

wasm-pack build

test: npm run start
